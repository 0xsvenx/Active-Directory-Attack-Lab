# Active Directory Attack Lab

This project demonstrates a full attack path against a vulnerable Active Directory environment,
starting from initial access to full Domain Admin compromise.

## Scope
- Internal corporate network
- Single domain forest
- Misconfigured AD services

## Tools Used
- Nmap
- CrackMapExec
- Impacket
- BloodHound
- Mimikatz

## Attack Path Overview
1. Network Enumeration
2. SMB & LDAP Enumeration
3. Credential Harvesting
4. Privilege Escalation
5. Lateral Movement
6. Domain Admin Compromise

## Outcome
Successfully achieved Domain Admin access and documented remediation steps.
