# Active Directory Attack Lab

This project demonstrates a full attack path against a vulnerable Active Directory environment,
starting from initial access to full Domain Admin compromise.

## ⚠️ Disclaimer
This project is for educational and ethical penetration testing purposes only.
All techniques demonstrated were performed in a controlled lab environment.

## Scope
- Internal corporate network
- Single domain forest
- Misconfigured AD services

## Tools Used
- Nmap
- CrackMapExec
- Impacket
- BloodHound
- Mimikatz

## Skills Demonstrated
- Active Directory enumeration and attack paths
- SMB, LDAP, and Kerberos abuse
- Credential harvesting and reuse
- Privilege escalation in Windows domains
- Lateral movement techniques
- Domain persistence awareness
- Security misconfiguration analysis

## Attack Path Overview
1. Network Enumeration
2. SMB & LDAP Enumeration
3. Credential Harvesting
4. Privilege Escalation
5. Lateral Movement
6. Domain Admin Compromise

## Outcome
Successfully achieved Domain Admin access and provided remediation recommendations aligned with enterprise security best practices.
